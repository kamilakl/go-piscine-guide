## Условия задания 

Эта программа помогает потренироваться в работе с аргументами командной строки в Go.  

| **Instruction** | **Перевод** |
|---------------------------|--------------|
| *Write a **program** that prints the arguments received in the command line.* | *Напишите **программу**, которая выводит аргументы, полученные в командной строке.* |

---

Нужно написать функцию, которая **выводит аргументы**- то есть все что прописано после имя файла по списку.

---

### Пример

- **Ввод:** `$ go run . choumi is the best cat`
- **Вывод:**
```
choumi
is
the
best
cat
```

*программа компилирует и сразу исполняет функций прописанные в файлах этой попки/директории на аргументы в консоле*

**Что нужно запомнить:**
- `go run` комипилирует файлы. Очень полезно при  тестировке файлов
- `.` указание на текущую директорию и компилирование все файлы в них с определненным раширением

---

## Решение

```go
package main

import (
	"os"
	"github.com/01-edu/z01"
)

func main() {
	// Проходим по аргументам, начиная с первого (os.Args[0] — это имя программы)
	for _, arg := range os.Args[1:] {
		// Для каждого аргумента выводим его символы по одному
		for _, runeChar := range arg {
			z01.PrintRune(runeChar) // Печатаем каждый символ аргумента
		}
		z01.PrintRune('\n') // После каждого аргумента выводим новую строку
	}
}

```

## Ключевые понятия

| **Инструмент / элемент кода** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **Пакет (`package main`)** | Объявляет, что программа является исполняемой. Все программы Go начинаются с `package main`.|
| **Импорт (`import (...)`)** | Позволяет использовать внешние пакеты. Здесь импортируются стандартный пакет `os` и учебный пакет `github.com/01-edu/z01` для вывода символов. |
| **Функция `main()`** | Главная точка входа программы. Код внутри неё выполняется при запуске. |
| **Цикл иттерации for...range**| Перебирает по всему что есть который указан в цикле(1 цикл аргументы, 2 символы) |
| **os.Args[1:]** | Это срез строк который содержит все аргументы после имени программы |
| **z01.PrintRune** | Функция из библиотеки `z01`, которая выводит одну руну (символ) на экран. Используется вместо `fmt.Print`. |
| **z01.PrintRune('\n')** | Выводит символ новой строки |
