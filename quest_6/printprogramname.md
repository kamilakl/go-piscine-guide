## Условия задания 

Эта программа помогает потренироваться в работе с аргументами командной строки в Go.  

| **Instruction** | **Перевод** |
|---------------------------|--------------|
| *Write a program that prints the name of the program.* | *Напишите программу, которая выводит название программы.* |

---

Нужно написать функцию, которая **выводит имя самого себя**- то есть имя программы, с которым она была скомпилирована.

---

### Примеры

- **Ввод:** `go build main.go`
- **Ввод:** `./main`
- **Вывод:** `main`

*программа компилирует и выводит то что прописанно "./" вводе*

- **Ввод:** `go build -o Nessy`
- **Ввод:** `./Nessy | cat -e`
- **Вывод:** `Nessy$`
  
*меняет имя программы через флаг -о и выводит имя с знаком `$`*

**Что нужно запомнить:**
- `go build` комипилирует файлы в папке в исполнимый файл
- `-o` меняет имя программы/файла
- `|` паип
- `cat -e`выводить все с окончанием на знак `$`

---

## Решение

```go
package main

import (
	"os"
	"github.com/01-edu/z01"
)

func main() {
	// Получаем имя программы из os.Args[0]
	programName := os.Args[0]

	// Проходим по каждому символу имени программы и выводим его с помощью z01.PrintRune
	for i := 0; i < len(programName); i++ {
		z01.PrintRune(rune(programName[i])) // Конвертируем каждый байт в руну и выводим
	}
	z01.PrintRune('\n') // Выводим символ новой строки
}

```

## Ключевые понятия

| **Инструмент / элемент кода** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **Пакет (`package main`)** | Объявляет, что программа является исполняемой. Все программы Go начинаются с `package main`.|
| **Импорт (`import (...)`)** | Позволяет использовать внешние пакеты. Здесь импортируются стандартный пакет `os` и учебный пакет `github.com/01-edu/z01` для вывода символов. |
| **Функция `main()`** | Главная точка входа программы. Код внутри неё выполняется при запуске. |
| **Aргумент (`os.Args[0]`)**| Срез аргументов (`os.Args`)** |
| **Цикл for** | Перебирает по длине `len(programName)` который указан в цикле по 1 символу |
| **z01.PrintRune** | Функция из библиотеки `z01`, которая выводит одну руну (символ) на экран. Используется вместо `fmt.Print`. |
| **z01.PrintRune('\n')** | Выводит символ новой строки |
