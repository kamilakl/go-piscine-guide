## Пример

**Ввод:**
```go
piscine.PrintNbrInOrder(321)
piscine.PrintNbrInOrder(0)
piscine.PrintNbrInOrder(321)
```

**Вывод:**
```
123
0
123
```

---

## Решение

```go
package piscine

import "github.com/01-edu/z01"

func PrintNbrInOrder(n int) {
	if n == 0 {
		z01.PrintRune('0')
		return
	}

	var digits []rune
	for n > 0 {
		digits = append(digits, rune(n%10)+'0')
		n /= 10
	}

	for i := 0; i < len(digits); i++ {
		for j := i + 1; j < len(digits); j++ {
			if digits[j] < digits[i] {
				digits[i], digits[j] = digits[j], digits[i]
			}
		}
	}

	for _, r := range digits {
		z01.PrintRune(r)
	}
}
```

## Логика и пошаговый разбор кода

### 1. Проверка на ноль

```go
if n == 0 {
	z01.PrintRune('0')
	return
}
```

- Если число равно нулю, сразу печатаем `'0'` и выходим.  
- Нет смысла дальше разбирать его на цифры и сортировать.

---

### 2. Разбор числа на цифры

```go
var digits []rune
for n > 0 {
	digits = append(digits, rune(n%10)+'0')
	n /= 10
}
```

- Создаём пустой срез `digits`, в который будем добавлять цифры.  
- `n % 10` — последняя цифра числа.  
- `rune(n%10) + '0'` — превращает цифру в символ.  
- `append` добавляет элемент в конец среза.  
- `n /= 10` — “отрезает” последнюю цифру числа.

**Пример:**  
`n = 421`  
→ `n%10 = 1`, добавляем `'1'`  
→ `n = 42`, `n%10 = 2`, добавляем `'2'`  
→ `n = 4`, `n%10 = 4`, добавляем `'4'`  
→ получаем `digits = ['1', '2', '4']` (в обратном порядке).

---

### 3. Сортировка цифр по возрастанию

```go
for i := 0; i < len(digits); i++ {
	for j := i + 1; j < len(digits); j++ {
		if digits[j] < digits[i] {
			digits[i], digits[j] = digits[j], digits[i]
		}
	}
}
```

- Двойной цикл сравнивает каждую цифру со всеми последующими.  
- Если находим меньшую цифру — меняем их местами.  
- Это простая реализация **сортировки выбором (Selection Sort algorithm)**.  

**Пример:**
```
digits = ['4','2','1']
i=0, j=1 → '2'<'4' → ['2','4','1']
i=0, j=2 → '1'<'2' → ['1','4','2']
i=1, j=2 → '2'<'4' → ['1','2','4']
```
Результат: `'1','2','4'`.

---

### 4. Печать цифр в правильном порядке

```go
for _, r := range digits {
	z01.PrintRune(r)
}
```

- Перебираем все символы в срезе `digits`.  
- `r` — каждая цифра в виде символа.  
- `PrintRune(r)` выводит цифру на экран.

Пример:  
`digits = ['1','2','4']`  
→ вывод: `124`.

## Почему нельзя использовать `int64`

- Задание специально ограничивает решение типом `int`.  
- Преобразование в `int64` здесь не нужно — диапазона `int` достаточно.  
- Это тренирует умение работать с типами данных и не усложнять задачу без причины.

## Заключение

Функция `PrintNbrInOrder`:
- Принимает целое число (кроме отрицательных);  
- Разбивает его на цифры;  
- Сортирует цифры по возрастанию;  
- И печатает их по порядку на экран.  

Это задание помогает понять:
- Как работать с числами и их разбором на цифры;  
- Как хранить символы в срезах;  
- Как реализовать простую **сортировку**;  
- И как использовать `rune` для вывода символов.
