# Quest 5.7 Concat

## Название файла Go
`concat.go`

---

## Условия задания 

| **Instruction** | **Перевод** |
|---------------------------|--------------|
| Write a function that returns the concatenation of two strings passed in arguments. | Напиши функцию, которая возвращает объединение (конкатенацию) двух строк, переданных в аргументах. |

---

Нужно написать функцию, которая **соединяет две строки** и возвращает результат в виде новой строки.

---

### Пример

- **Ввод:**
  ```go
  fmt.Println(piscine.Concat("Hello!", " How are you?"))
  ```
- **Вывод:**
  ```
  Hello! How are you?
  ```

Дополнительно:
- `Concat("H", "olla")` → `Holla`
- `Concat("Hello", " world!")` → `Hello world!`

---

## Решение

```go
package piscine

func Concat(str1 string, str2 string) string {
	return str1 + str2
}
```

---

## Ключевые понятия

| **Инструмент / элемент кода** | **Объяснение и назначение** |
|-------------------------------|------------------------------|
| **Оператор `+` для строк** | В Go используется для **конкатенации** (соединения) строк. Например, `"Go" + "lang"` = `"Golang"`. |
| **Тип `string`** | Неизменяемая последовательность символов Unicode. В Go строки являются **immutable** — при объединении создаётся новая строка. |
| **Неизменяемость (immutability)** | В Go нельзя изменить уже созданную строку. Каждое объединение создаёт новый объект в памяти. |
| **Эффективность** | При множественной конкатенации (например, в цикле) лучше использовать `strings.Builder`, чтобы избежать лишних выделений памяти. |

---

## Логика и пошаговый разбор кода

### 1. Объявление функции

```go
func Concat(str1 string, str2 string) string {
```

Функция принимает два аргумента:
- `str1` — первая строка;
- `str2` — вторая строка.  
Тип возвращаемого значения — также `string`.

---

### 2. Конкатенация строк

```go
return str1 + str2
```

- Оператор `+` соединяет строки.  
- Создаётся **новая строка**, состоящая из `str1`, к которой добавляется `str2`.

Пример:
```go
Concat("Hello", "World") → "HelloWorld"
```

---

## Важно знать

> **Строки в Go неизменяемы.** После создания строка не может быть изменена.  
> Когда выполняется `str1 + str2`, в памяти создаётся **новая строка**, содержащая все символы первой строки, за которыми следуют символы второй.


> Для более эффективного объединения множества строк можно использовать пакет `strings` и тип `strings.Builder`:
```go
var builder strings.Builder
builder.WriteString("Go")
builder.WriteString("lang")
fmt.Println(builder.String()) // Golang
```
