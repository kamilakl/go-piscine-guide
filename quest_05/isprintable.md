# Quest 5.11 Is Printable

## Название файла Go
`printable.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| Write a function that returns `true` if the string passed as a parameter contains only printable characters, otherwise returns `false`. | Написать функцию, которая возвращает `true`, если строка содержит только печатные символы. |

---

Нужно написать функцию, которая проверяет строку:  
если она содержит **только печатаемые ASCII-символы**, возвращается `true`, иначе `false`.

---

### Пример

- **Ввод:**  
  `"Hello"`  
  `"Hello\n"`  
  `"abc123!"`  
  `"Привет"`
- **Вывод:**  
  `true`  
  `false`  
  `true`  
  `false`

---

## Решение

```go
package piscine

func IsPrintable(s string) bool {
	for _, r := range s {
		if r < 32 || r > 126 {
			return false
		}
	}
	return true
}
```

---

## Ключевые понятия

| Элемент | Объяснение |
|--------|------------|
| **`rune`** | Числовой Unicode-код символа. Позволяет проверять диапазоны. |
| **Диапазон 32–126** | Все печатные символы ASCII: пробел, буквы, цифры, знаки. Всё что `<32` — управляющие символы (`\n`, `\t`, `\r`). |
| **Цикл `for range`** | Перебирает строку по рунам, что важно для корректной проверки кодов символов. |

---

## Логика и пошаговый разбор кода

### 1. Перебор символов строки
```go
for _, r := range s {
```
Получаем каждую руну (символ) строки.

### 2. Проверка печатаемости
```go
if r < 32 || r > 126 {
	return false
}
```
Если символ вне ASCII-диапазона печатных символов возвращаем `false`.

### 3. Возврат результата
Если цикл закончился не обнаружив запрещенных символов возвращаем `true`.

---

## Заключение

Функция **IsPrintable** проверяет, состоит ли строка только из печатных ASCII-символов.  
Подходит для валидации пользовательского ввода, фильтрации данных и защиты от скрытых символов.
