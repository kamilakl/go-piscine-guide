# Quest 5.19 Join

## Название файла Go
`join.go`

---

## Условия задания 

| **Instruction** | **Перевод** |
|-----------------|-------------|
| Write a function that returns the concatenation of all the strings of a slice of strings separated by the separator passed as the argument `sep`. | Напиши функцию, которая возвращает объединённую строку из всех элементов среза строк, вставляя между ними разделитель `sep`. |

---

Функция должна:
- принять срез строк (`[]string`) и строку-разделитель (`sep`),
- склеить все строки в одну,
- поставить `sep` между каждым элементом,
- корректно обработать пустой срез.

---

### Пример

**Ввод:**
```go
fmt.Println(piscine.Join([]string{"apple", "banana", "cherry"}, ", "))
```

**Вывод:**
```
apple, banana, cherry
```

---

## Решение

```go
package piscine

func Join(strs []string, sep string) string {
    if len(strs) == 0 {
        return ""
    }

    result := strs[0]

    for i := 1; i < len(strs); i++ {
        result += sep + strs[i]
    }

    return result
}
```
---

## Ключевые понятия

| **Элемент** | **Объяснение** |
|-------------|----------------|
| **`[]string`** | Срез строк — динамический список, в котором можно хранить любое количество элементов. |
| **Конкатенация строк** | Процесс склеивания строк друг с другом |
| **Перебор среза** | Цикл `for` используется, чтобы пройти по каждому элементу. |
| **`len(strs)`** | Возвращает длину среза, чтобы знать, сколько элементов нужно объединить. |

---

## Логика и пошаговый разбор кода

### 1. Проверка на пустой срез
```go
package piscine

func Join(strs []string, sep string) string {
    if len(strs) == 0 {
        return ""
    }
```
Если список пуст, возвращаем пустую строку.  
Это защищает от ошибок и делает поведение предсказуемым.

---

### 2. Начальное значение результата
```go
    result := strs[0]
```
Берём первый элемент как основу итоговой строки.  
Это позволяет **не добавлять разделитель перед первым элементом**.

---

### 3. Цикл по оставшимся строкам
```go
    for i := 1; i < len(strs); i++ {
        result += sep + strs[i]
    }
```

На каждой итерации:
- добавляем разделитель `sep`,
- затем строку `strs[i]`.

Короткая демонстрация:

| i | result |
|---|--------|
| start | `"apple"` |
| 1 | `"apple, banana"` |
| 2 | `"apple, banana, cherry"` |

---

### 4. Возврат результата
```go
    return result
```

Итоговая строка возвращается после полного прохода по срезу.

## Заключение
Функция Join полезна для работы с коллекциями строк, когда нужно объединить их в одну строку с заданным разделителем (отличие от задания `BasicJoin`). 
Важно понимать, как работают срезы, циклы и конкатенация строк в Go. 
Этот код поможет вам эффективно соединять строки, а также учить работать с основными структурами данных Go.
