# Quest 5.14 To Lower

## Название файла Go
`tolower.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| Write a function that turns every letter of the string into lowercase. | Написать функцию, которая переводит каждую букву строки в нижний регистр. |

---

Нужно написать функцию, которая принимает строку и возвращает новую строку, где все заглавные буквы `'A'`–`'Z'` преобразованы в строчные `'a'`–`'z'`, а остальные символы остаются без изменений.

---

### Пример

- **Ввод:**
  ```go
  fmt.Println(piscine.ToLower("Hello! HOW Are YOU?"))
  ```
- **Вывод:**
  ```
  hello! how are you?
  ```

---

## Решение

```go
package piscine

func ToLower(s string) string {
	result := ""
	for _, r := range s {
		if r >= 'A' && r <= 'Z' {
			r = r + 32
		}
		result += string(r)
	}
	return result
}
```

---

## Ключевые понятия

| **Термин или инструмент** | **Объяснение** |
|--------|------------|
| **Диапазон `'A'`–`'Z'`** | Заглавные латинские буквы (ASCII 65–90). |
| **Преобразование в нижний регистр** | Разница между `'A'` (65) и `'a'` (97) всегда **32**, поэтому `r + 32` превращает символ в строчный. |
| **Цикл `for range`** | Перебирает строку по Unicode-символам (`rune`). |
| **Сбор строки через `+=`** | Создаёт новую строку, добавляя в неё преобразованные символы. |
