# Quest 5 — IsAlpha

## Название файла Go
`isalpha.go`  

---

## Условия задания

| **Instruction** | **Перевод** |
|------------------|-------------|
| *Write a function that returns `true` if the string passed as the parameter only contains alphanumerical characters or is empty, otherwise, and returns `false.`* | *Напишите функцию, которая возвращает `true`, если переданная строка содержит только буквенно-цифровые символы (буквы и цифры) или является пустой. В противном случае функция возвращает `false`.* |

---

## Решение

```go
package piscine

func IsAlpha(s string) bool {
	for _, letter := range s {
		if (letter < 'A' || letter > 'Z') &&
			(letter < 'a' || letter > 'z') &&
			(letter < '0' || letter > '9') {
			return false
		}
	}
	return true
}
```

---

## Примеры использования

```go
package main

import (
	"fmt"
	"piscine"
)

func main() {
	fmt.Println(piscine.IsAlpha("Hello! How are you?")) // false
	fmt.Println(piscine.IsAlpha("HelloHowareyou"))      // true
	fmt.Println(piscine.IsAlpha("What's this 4?"))      // false
	fmt.Println(piscine.IsAlpha("Whatsthis4"))          // true
	fmt.Println(piscine.IsAlpha(""))                    // true (пустая строка — допустима)
}
```

## Логика и пошаговый разбор кода

### 1. Объявление функции

```go
func IsAlpha(s string) bool {
```

Принимает строку `s` и возвращает `true` или `false`.  
Функция проверяет состоит ли строка только из букв и цифр, либо пуста.

---

### 2. Перебор всех символов строки

```go
for _, letter := range s {
```

- Цикл `for range` проходит по каждому символу (`rune`) в строке.  
- `_` — это placeholder для индекса, обозначение значит что индекс в цикле нам не нужен.  
- `letter` хранит числовое значение символа.

---

### 3. Проверка каждого символа

```go
if (letter < 'A' || letter > 'Z') &&
    (letter < 'a' || letter > 'z') &&
	(letter < '0' || letter > '9') {
	return false
}
```

- Если символ находится вне допустимых диапазонов (`'A'–'Z'`, `'a'–'z'`, `'0'–'9'`),  
  функция сразу возвращает `false`.  

  - Под вне диапазона здесь подразумеваем:  
  если символ меньше ASCII значения `'A'` или больше `'Z'` (аналогично для других диапазонов). 
  Поэтому в условиях используется логическое ИЛИ: если хотя бы одно условие верно, символ не подходит.  

- Как только найден хотя бы один запрещённый символ (например `'!'`), функция немедленно возвращает `false`.

---

### 4. Возврат итогового значения

```go
return true
```

- Если цикл завершился без преждевременного выхода значит все символы допустимы.  
- Даже если строка была пустой, она автоматически считается валидной.

## Визуализация проверки

Пример:  
`s = "Go123!"`

| Символ | Проверка диапазонов | Результат |
|---------|--------------------|------------|
| `'G'` | входит в `'A'–'Z'` | Пройдена проверка |
| `'o'` | входит в `'a'–'z'` | Пройдена проверка |
| `'1'` | входит в `'0'–'9'` | Пройдена проверка |
| `'2'` | входит в `'0'–'9'` | Пройдена проверка |
| `'3'` | входит в `'0'–'9'` | Пройдена проверка |
| `'!'` | не входит ни в один диапазон | `return false` |
