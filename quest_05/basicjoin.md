# Quest 5.18 Basic Join

## Название файла Go
`basicjoin.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| Write a function that returns a concatenated string from the 'strings' passed as arguments. | Напишите функцию, которая возвращает строку, полученную путём конкатенации всех строк, переданных в аргументах (в виде среза). |

---

Функция должна принять **срез строк** и вернуть одну строку, склеив все элементы подряд

---

### Пример

**Ввод:**
```go
fmt.Println(piscine.BasicJoin([]string{"she", "went", "home"}))
```

**Вывод:**
```
shewenthome
```

---

## Решение

```go
package piscine

func BasicJoin(elems []string) string {
    result := ""
    for _, str := range elems {
        result += str
    }
    return result
}
```

---

## Ключевые понятия

| **Элемент** | **Объяснение** |
|-------------|----------------|
| **`string`** | Тип данных, представляющий строку. В Go строки неизменяемы: изменить символ по индексу нельзя, можно только создать новую строку. |
| **`[]string`** | Срез строк. Динамическая структура, позволяющая хранить список строк любой длины. |
| **`for _, str := range elems`** | Цикл `for range` перебирает элементы среза. `_` — игнорируем индекс, нам нужна только строка `str`. |
| **Конкатенация** | Склеивание строк друг с другом. В Go используется оператор `+`, например: `result += str`. |

---

## Логика и пошаговый разбор кода

### 1. Объявление функции

```go
package piscine

func BasicJoin(elems []string) string {
```

Функция принимает срез строк `elems` и возвращает одну строку.

---

### 2. Создание строки для хранения результата

```go
    result := ""
```

Пустая строка, в которую мы будем постепенно добавлять все элементы.

---

### 3. Цикл по элементам среза

```go
for _, str := range elems {
    result += str
}
```

- `range elems` перебирает каждый элемент внутри среза.  
- Переменная `str` хранит текущий элемент.  
- `result += str` добавляет новую строку в конец.

Каждая итерация «удлиняет» результат.

Короткая демонстрация:

| elems | result |
|-------|--------|
| `"she"` | `"she"` |
| `"went"` | `"shewent"` |
| `"home"` | `"shewenthome"` |

---

### 4. Возврат итоговой строки

```go
return result
```

## Заключение

Функция `BasicJoin` демонстрирует простую, но важную операцию как соединение строк из списка в одну строку. 
Это базовый навык работы со срезами, циклами и строками. Он пригодится при:
- сборке текста из частей,
- обработке данных,
- создании собственных функций форматирования,
- работе с массивами или пользовательским вводом.
