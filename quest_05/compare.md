---

Нужно написать функцию, которая **сравнивает две строки** и возвращает результат сравнения в виде числа:
- `0`, если строки равны;
- `-1`, если первая строка меньше второй;
- `1`, если первая строка больше второй.

---

## Пример выполнения

| **Выражение** | **Результат** |
|----------------|----------------|
| `"abc" == "abc"` | `0` |
| `"abc" < "abd"` | `-1` |
| `"abd" > "abc"` | `1` |

---

## Решение

```go
package piscine

func Compare(a, b string) int {
	if a == b {
		return 0
	} else if a < b {
		return -1
	}
	return 1
}
```

## Логика и пошаговый разбор кода

### 1. Объявление функции
```go
func Compare(a, b string) int {
```
Функция принимает две строки `a` и `b`, и возвращает одно число (`int`).

---

### 2. Проверка на равенство
```go
    if a == b {
	    return 0
    }
```
Если строки совпадают посимвольно — возвращаем `0`.  
Пример: `"Hello"` и `"Hello"` → `0`.

---

### 3. Проверка на "меньше"
```go
    else if a < b {
	    return -1
    }
```
Если первая строка идёт раньше второй в лексикографическом порядке, возвращаем `-1`.  
Пример: `"Apple"` меньше `"Banana"` → `-1`.

---

### 4. Иначе — первая строка больше
```go
    return 1
```
Если не равна и не меньше — значит больше.  
Пример: `"Zebra"` больше `"Apple"` → `1`.

---

 Мыс сравниваем строки именно **посимвольно**, используя Unicode (ASCII) значения символов.

Например:  
| Символ | Код ASCII |
|---------|------------|
| `'A'` | 65 |
| `'B'` | 66 |
| `'a'` | 97 |
| `'b'` | 98 |

`"Hello"` < `"World"`, потому что `'H'` (72) < `'W'` (87).

## Заключение

Функция **Compare** реализует простое **лексикографическое сравнение строк**:
- Возвращает `0`, если строки равны;
- Возвращает `-1`, если первая строка меньше;
- Возвращает `1`, если первая строка больше.

Эта функция аналогична стандартной функции `strings.Compare()` из пакета `strings` в Go.  
Она полезна, когда нужно определить порядок строк, например, при сортировке или поиске.
