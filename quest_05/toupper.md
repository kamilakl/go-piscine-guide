# Quest 5.13 To Upper

## Название файла Go
`toupper.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| Write a function that makes every letter of the string uppercase. | Написать функцию, которая делает все буквы строки заглавными. |

---

Нужно создать функцию, которая принимает строку и возвращает новую строку, где *все строчные буквы `'a'`–`'z'` преобразованы в заглавные `'A'`–`'Z'`.  
Все остальные символы (цифры, пробелы, знаки, Unicode-символы) остаются без изменений.

---

### Пример

- **Ввод:**
  ```go
  fmt.Println(piscine.ToUpper("Hello! How are you?"))
  ```
- **Вывод:**
  ```
  HELLO! HOW ARE YOU?
  ```

---

## Решение

```go
package piscine

func ToUpper(s string) string {
	result := ""
	for _, r := range s {
		if r >= 'a' && r <= 'z' {
			r = r - 32
		}
		result += string(r)
	}
	return result
}
```

---

## Ключевые понятия

| **Элемент** | **Объяснение** |
|--------|------------|
| **Диапазон `'a'`–`'z'`** | Строчные латинские буквы (ASCII 97–122). |
| **Разница между `'a'` и `'A'` — 32** | В ASCII `'a'` (97) и `'A'` (65) всегда различаются на `32`. Поэтому `r - 32`, где `r` - ASCII код текущего символа, превращает букву в заглавную. |
| **Цикл `for range`** | Перебирает строку по рунам, что позволяет корректно работать с Unicode. |
| **Конкатенация `result += string(r)`** | Добавляет текущий символ (уже преобразованный или нет) в итоговую строку. |
| **Uppercase (верхний регистр)** | Заглавные буквы. |
| **Lowercase (нижний регистр)** | Строчные буквы. |

## Заключение

Функция **ToUpper** преобразует все строчные латинские буквы в заглавные, используя разницу в 32 между ASCII кодами заглавных и строчных букв.  
Все остальные символы остаются неизменными.

**Что запомнить:**
- Диапазон `'a'`–`'z'` → строчные  
- `'A'`–`'Z'` → заглавные  
- `r - 32` — быстрый способ перевести букву в верхний регистр  
- Строки в Go неизменяемы
