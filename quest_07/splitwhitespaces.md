# Quest 7.4 - SplitWhiteSpaces

## Название файла Go
`splitwhitespaces.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| *Write a function that separates the words of a `string` and puts them in a string slice. The separators are spaces, tabs and newlines.* | *Напишите функцию, которая разделяет строку на слова и возвращает их в виде среза строк. Разделителями считаются пробелы, табуляции и символы новой строки.* |

---

### Пример использования

```go
package main

import (
	"fmt"
	"piscine"
)

func main() {
	fmt.Printf("%#v\n", piscine.SplitWhiteSpaces("Hello how are you?"))
}
```

Вывод:
```
[]string{"Hello", "how", "are", "you?"}
```

---

## Решение

```go
package piscine

func SplitWhiteSpaces(s string) []string {
	var result []string
	word := ""

	for i := 0; i < len(s); i++ {
		// Проверяем разделители: пробел, таб, перевод строки
		if s[i] == ' ' || s[i] == '\t' || s[i] == '\n' {
			// Добавляем в результат собранное слово
			if word != "" {
				result = append(result, word)
				word = ""
			}
		} else {
			// Строим слово посимвольно
			word += string(s[i])
		}
	}

	// Финальное слово (если было) тоже добавляем
	if word != "" {
		result = append(result, word)
	}

	return result
}
```

## Ключевые понятия

| **Инструмент / концепт** | **Объяснение** |
|--------------------------|----------------|
| **Срез (`[]string`)** | Хранит набор строк. |
| **append()** | Добавляет найденное слово в конец итогового списка. Изучите как именно `append()` это делает. |

### Что такое "white spaces"?
К пробельным символам относятся:

- `' '` — пробел,
- `'\t'` — табуляция,
- `'\n'` — перевод строки.

Они часто используются как разделители текста.

### Как работает ручной сплит?
В отличие от `strings.Fields`, в Piscine нельзя использовать стандартную библиотеку.  
Поэтому мы вручную:
1. перебираем строку по символам;
2. собираем буквы в переменную;
3. как только встречаем разделитель закрываем слово и добавляем в срез;
4. продолжаем так до конца строки.

## Логика и пошаговый разбор кода

### 1. Подготовка переменных

```go
package piscine

func SplitWhiteSpaces(s string) []string {
	var result []string
	word := ""
```
- `result` будет хранить все слова,
- `word` будет временно накапливать символы.


### 2. Посимвольный проход по строке

```go
    for i := 0; i < len(s); i++ 
```

В данном решении мы проходимся по байтам - **попробуйте в качестве практики написать перебор по рунам, он здесь более безопасен и уместен.** 

### 3. Проверка: является ли символ разделителем? 

```go
        if s[i] == ' ' || s[i] == '\t' || s[i] == '\n' {
            if word != "" {
	        result = append(result, word)
	        word = ""
        }
```
Если да, значит слово закончилось и можно добавлять его в срез. А `word` снова очистить чтобы начать собирать следующее слово.
Важно: мы добавляем слово только если оно не пустое (например, в строке могло быть несколько пробелов подряд)

---

### 5. Если символ не является разделителем

```go
        } else {
			word += string(s[i])
		}
    }
```
Если нет, это значит мы перебираем буквы. Добавляем их к `word` чтобы сформировать текущее слово.

---

### 6. После цикла добавляем последнее слово

```go
    if word != "" {
	result = append(result, word)
}
```
Если строка заканчивается словом, без пробела после него, то условие в цикле не сработает и слово просто не попадёт в результат. Поэтому последнее слово добавляем уже вне цикла.

## Заключение
Функция `SplitWhiteSpaces`:
- вручную разделяет строку на слова,
- корректно обрабатывает несколько пробелов,
- игнорирует пустые слова,
- работает без `fmt`, `strings` и прочих пакетов.

Это важная практика, дающая понимание работы со строками и обработки символов.
