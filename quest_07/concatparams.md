# Quest 7.3 — ConcatParams

## Название файла Go
`concatparams.go`

---

## Условия задания

| **Instruction** | **Перевод** |
|-----------------|-------------|
| *Write a function that takes the arguments received in parameters and returns them as a `string`. The `string` is the result of all the arguments concatenated with a newline (`\n`) between.* | *Напишите функцию, которая принимает срез строк и возвращает одну строку, где все элементы объединены между собой с помощью символа новой строки (`\n`).* |

---

Нужно написать функцию, которая:

- принимает `[]string`,
- соединяет все элементы в одну строку,
- между каждым элементом вставляет `\n`,
- не добавляет лишний перевод строки в конце.

### Пример использования

```go
package main

import (
	"fmt"
	"piscine"
)

func main() {
	test := []string{"Hello", "how", "are", "you?"}
	fmt.Println(piscine.ConcatParams(test))
}
```

Вывод:
```
Hello
how
are
you?
```

---

## Решение

```go
package piscine

func ConcatParams(args []string) string {
	length := len(args)

	// при пустом срезе выводим пустую строку
	if length == 0 {
		return ""
	}

	result := ""

	for i := 0; i < length; i++ {
		result += args[i]

		// Добавляем \n только если элемент НЕ последний
		if i != length-1 {
			result += "\n"
		}
	}

	return result
}
```

---

## Ключевые понятия

| **Инструмент / концепт** | **Объяснение** |
|--------------------------|----------------|
| **Срез (`[]string`)** | Структура данных, содержащая динамический список строк. |
| **Конкатенация строк (`+=`)** | Объединение строк. В Go создаёт новую строку в памяти. |

В Go строки **immutable**. Их нельзя изменить после создания. А при каждом `+=` будет создается новая строка.

Здесь решение с `+=` учебное и простое, но не оптимальное для больших текстов.  
В реальных проектах используют более оптимальный (и запрещенный в задании) `strings.Builder` не создающий копий.

**Полезно почитать:**  
- https://metanit.com/go/tutorial/2.13.php - о срезах строк
- https://pkg.go.dev/strings#Builder - `strings.Builder`

## Логика и пошаговый разбор кода

### 1. Проверяем, пустой ли срез

```go
package piscine

func ConcatParams(args []string) string {
	length := len(args)
    if length == 0 {
	    return ""
    }
```

Сохраняем `len(args)` в переменную чтобы не вызывать ее без конца. Если входных строк нет, возвращаем так же `""`.

---

### 2. Инициализируем переменную для хранения результата и цикл для перебора

```go
    result := ""
    for i := 0; i < length; i++ {
	    result += args[i]
        if i != length-1 {
	        result += "\n"
        }
    }
    return result

```
Цикл проходит по каждому элементу среза `args` начиная с нулевого индекса до индекса `args[length-1]` включительно, обновляя `result`

Каждую итерацию проверяется на условие:

- если элемент под индексом `i` **не последний**, добавляем перевод строки,
- если последний то не добавляем чтобы не получить лишний пустой ряд.

В конце возвращаем получившуюся строку.
